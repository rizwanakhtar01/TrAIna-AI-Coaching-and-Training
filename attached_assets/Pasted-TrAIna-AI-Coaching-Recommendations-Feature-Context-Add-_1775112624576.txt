TrAIna — AI Coaching Recommendations Feature
Context
Add the AI Coaching Recommendations feature to the existing TrAIna prototype. This feature extends the Challenging Patterns Explorer with AI-generated coaching cards that supervisors review, approve, and assign to agents. Agents receive and complete coaching plans tracked in real time.
The existing app already has: Challenging Patterns Explorer (list view), Pattern Detail page with impacted agents table, and a Coaching Progress tab. This prompt covers the new screens and modifications needed.

Screens to Build (6 total)
Screen 1 — Enhanced Patterns Explorer (modify existing)
Add to each pattern card:

Trend indicator: arrow icon + percentage change badge (↑ Worsening/red, → Stable/amber, ↓ Improving/green)
Agent count alongside existing frequency and channel tags
Keep existing filters (time range, channel) and layout

Screen 2 — Pattern Detail Page (modify existing)
Keep existing: stat cards (occurrences, agents, channels, timeframe) and impacted agents table.
Add new section below impacted agents:

Green-bordered card titled "AI coaching recommendation"
Shows one coaching card per pattern with:

Pattern name (bold) + "Area to improve" subtitle
Status badge: "Ready for review" (green) or "Generating..." (amber)
Section count: "5 sections: Coaching objective, Talk tracks, Policy & KB, Practice scenario, Best practices"
"Review card →" button (primary blue) navigates to Screen 3

Practice scenario — purple bg card with the AI simulation prompt. Footer note: "Interactive practice available in Phase 2"
Best practices from top performers — bulleted list of 4-5 behavioral tips



Right column — Action sidebar (sticky):

"Approve card" button (green, full width, prominent)
"Edit sections" button (outline gray)
"Reject card" button (outline red)
Divider
"Assign to agents" section: checkboxes for impacted agents (pre-checked for agents above threshold, unchecked with note for those below)
"Due date" input (default: 7 days from today)

Aggregate progress bar + fraction

Table: Agent (avatar + name), Pattern, Area to improve, Status badge, Progress bar + fraction, Due date, Action button ("Nudge" for overdue)
